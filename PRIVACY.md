# 隐私政策

本隐私政策说明「雪哨（xueqiu-watch）」Chrome 扩展（以下简称"本扩展"）如何处理用户数据。

## 一、我们收集 / 使用哪些数据

1. **雪球登录态（间接使用）**
   本扩展通过你已在 Chrome 中打开并登录的 `xueqiu.com` 页面，在页面内调用雪球 API 来读取你「特别关注」分组中博主的时间线。该过程使用浏览器自动携带的登录 Cookie，**本扩展不读取、不存储你的雪球账号密码**。

2. **新帖内容**
   检测到的新帖标题 / 正文摘要，仅用于在本机弹窗、系统通知中向你展示；若你主动开启了「企业微信推送」，该内容会发送到**你自建的企业微信应用**。

3. **配置信息**
   轮询间隔、监控名单、声音开关等，仅用于本扩展运行，存储在本机。

## 二、数据存储位置

上述所有数据**仅存储在你本机浏览器的 `chrome.storage.local` 中**，不会上传到本扩展作者的任何服务器。

## 三、我们与谁共享数据

- 我们**不会**将你的任何数据出售或共享给广告商、数据分析商等第三方。
- **唯一例外**：当你主动开启「企业微信推送」时，新帖内容会发往**腾讯企业微信服务**（由你自己的企业微信账号和应用控制，数据归你控制，非本扩展作者可控）。

## 四、我们不会收集

- 不收集你的浏览器历史记录；
- 不收集 `xueqiu.com` 以外任何网站的数据；
- 不收集与「检测特别关注新帖」功能无关的个人信息；
- 不用于广告、不用于用户画像。

## 五、数据用途限定（符合 Chrome Web Store 2026-08-01 新规）

本扩展收集 / 使用的所有用户数据，**仅用于上述已披露的单一用途**——即检测你特别关注博主的新帖并向你本人发出提醒。我们不会将任何数据挪作他用，也不会在安装后擅自变更数据处理方式。

## 六、你的权利

你可以随时在扩展设置页清除运行日志、关闭推送、卸载扩展，相关数据即从本机删除。
